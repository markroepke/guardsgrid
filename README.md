# GuardsGrid

A 3×3 lineup game built around Cleveland Guardians/Indians franchise history. Pick a player who satisfies both the row and column criteria for each of the 9 cells. Inspired by Immaculate Grid.

## Play

Open `index.html` in any browser, or visit the deployed page.

## How it works

- All player data, criteria, and rendering live in a single self-contained `index.html` file — no build step, no dependencies.
- Roster covers ~400 unique players from 1901–2025.
- Criteria mix decades, awards, positions, handedness, and curated stat thresholds (200+ HR, 20-win season, no-hitters, etc.).

## Theme

Visual design takes cues from the Cleveland Guardians City Connect uniforms — navy, cream, and red, with art-deco bridge motifs referencing the Hope Memorial Bridge guardians.
